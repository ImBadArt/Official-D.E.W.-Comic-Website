# Official D.E.W. Comic Website (docs/)

This folder contains the deployable static site for GitHub Pages.

- All site files (HTML, JS, CSS, assets, comics, etc.) are here.
- Set GitHub Pages source to `main` branch, `/docs` folder.
- Add new images/chapters to `comics/official-dew-comic-website/chapters/`.
- Run `node generate-chapters-json.js` before each commit to update the manifest.
